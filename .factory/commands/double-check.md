Are you sure we did not forget something important? Think again. If you are sure we did not forget anything then tell it to me but be certain about it.

## Steps

1. **Re-read the original request.** Go back to the user's initial ask and any follow-up requirements.
2. **Inventory what was done.** List every change, file, or action taken.
3. **Cross-check against requirements.** For each requirement, confirm it is addressed. Flag any gap.
4. **Think about non-obvious misses.** Consider edge cases, tests, documentation, config changes, migrations, side effects, and anything that typically gets forgotten.
5. **Deliver your verdict.** If something was missed, say exactly what and propose a fix. If nothing was missed, state clearly that you are confident everything is covered and why.
